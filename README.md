## Overview

**Bongthom Vue** is a locally-focused classifieds and job board platform built for the Cambodian market. It provides a clean, accessible interface for browsing job listings, classified advertisements, and community announcements — all powered by Vue 3 with a static data architecture for fast, reliable performance.

Built with a mobile-first approach and Khmer-language friendly design.

## Key Features

- **Classified Listings** — Browse categorized ads with quick-view and detail modes
- **Job Board** — Company-based job listings with search and filtering capabilities
- **Responsive Sidebar** — Dynamic sidebar with quick links, RSS feeds, special schedules, and organization filters
- **Localized Design** — Khmer-friendly typography and culturally relevant UI patterns
- **Multi-language Ready** — Language bar component for future i18n expansion
- **Static Data Architecture** — JSON-driven content for fast loading without backend dependency

## Getting Started

### Prerequisites

- Node.js >= 20.19.0 or >= 22.12.0
- npm

### Installation

```bash
git clone https://github.com/OuThorninvithyea/bongthom_vue_js_v1.git
cd bongthom_vue_js_v1
npm install
npm run dev        # Dev server: http://localhost:5173
npm run build      # Production build
npm run preview    # Preview production build
```
